# Release Candidate FMI 3.0.1rc1 available for inspection

The release candidate 1 for FMI 3.0.1 is ready for final review: https://github.com/modelica/fmi-standard/releases/tag/v3.0.1-rc.1

According to the recently approved procedure for the release of maintenance release of MA standards, now a 4-week inspection period starts and will last until April 22. In the last week of the inspection period, only editorial or typographic changes are allowed
In case of findings, please raise issues on https://github.com/modelica/fmi-standard.
If there are no blockers detected, the FMI Project will release FMI 3.0.1 afterwards.
